¿Qué es el HTML5?


Es una combinación de nuevas etiquetas de HTML, propiedades de CSS3, JavaScript y de algunas tecnologías complementarías de apoyo. Toda esta mezcla es con el fin de mejorar la experiencia y simplificar los procesos que se han venido trabajando con las versiones anteriores del HTML.

Tema 2

Etiquetas Básicas, Atributos y Valores


* Etiquetas:


Las etiquetas en HTML se escriben con la siguiente formato:

< marca > contenido < /marca >


En en HTML hay muchas etiquetas emparejadas, es decir, que tienen un inicio y un cierre, como por ejemplo:

< h1 > (Inicio)
< /h1 > (Cierre)

Sin embargo, HTML cuenta a su vez con etiquetas sin emparejar, es decir, que terminan con una barra inclinada, como por ejemplo:

< hr / >

Como también permite que existan etiquetas sin necesidad de cierre ( < / >) como por ejemplo:

< img >

Ahora bien al saber que es una etiqueta y cómo está formada, vamos a presentar cuales son las etiquetas básicas esenciales para utilizar HTML

< header >:

Es la etiqueta que se utiliza para presentar la información introductoria del documento, que va ubicada en la cabecera del documento o en una sección, es por ello que pueden haber más de uno en el documento.

< footer >:


Es la etiqueta que se utiliza para contener la información sobre el autor (es), derechos, términos de uso, información de contacto, y se ubica en la parte final del documento.

< nav >:


Es la etiqueta que se usa para representar la sección que contiene los enlaces de navegación por el sitio web.

< section >:


Es la etiqueta que define una sección del documento.

< aside >:


Es la etiqueta que define el contenido secundario, por lo general es una barra lateral, y todo lo que contiene suele estar relacionado con el contenido principal.

< figure > y < figcaption >:


Son etiquetas que se usas para utilizar y definir un elemento independiente que guarda relación con el elemento que lo contiene; estos pueden ser una imagen, un fragmento de código, un diagrama, etc. < figcaption > se utiliza para adicionarle a este elemento independiente un título o una leyenda para identificarlo.

< details >:


Esta etiqueta se usa para ocultar una serie de detalles por parte del usuario.

*Atributos y Valores



Los atributos son los que especifican algunas características singulares de un elemento o información adicional. Los atributos se especifican como pares atributo="valor", separados por espacios dentro de la etiqueta de inicio:
< etiqueta atributo = "valor" > texto < / etiqueta>

En el siguiente ejemplo se presenta el uso del valor, el cual aunque no es obligatorio que vaya entre comillas, es recomendable usarlo de esa manera para mayor comprensión:

< a href = "http://www.ra-ma.es/" > Editorial RA-MA < / a >


Existen atributos genéricos, como el id = "texto", que se pueden utilizar para asociar a cualquier etiqueta. Esto es de gran ayuda al momento de utilizar CSS y JavaScript pues facilita mucho la comprensión entre ellas y evita confusiones.
Al tener esta ventaja también podemos asignar identificadores únicos a elementos dentro de la página tales como class = "texto", para así poder modificar y manipular dicho elemento sin necesidad de afectar a otros. También a dicho conjunto de elementos de la misma clase los podemos modificar de manera conjunta utilizando style = "texto", además al colocarle un título al elemento se logra mejorar la accesibilidad ya que los navegadores lo muestran cuando el usuario pasa el ratón por encima del elemento, esto se logra utilizando title = "texto"
